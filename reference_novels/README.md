# Reference novels directory

Put `.txt` or `.md` novels here. The reference demo will read whole files, split them into chunks, embed each chunk, retrieve relevant chunks, and ask the writer model to create a new story inspired by broad structure/atmosphere.

Notes:
- Use works you have rights to process, public-domain texts, or your own drafts.
- The demo is designed to borrow structure, mood, clue weaving, pacing, and motif usage, not to copy protected prose.
